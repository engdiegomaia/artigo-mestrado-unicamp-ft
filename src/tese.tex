\begin{resumo}
Este trabalho apresenta uma metodologia para classificação de uso e cobertura da terra (LULC) utilizando imagens hiperespectrais coletadas por drones, com foco em aplicações de agricultura de precisão. A pesquisa aborda os desafios inerentes ao processamento de dados hiperespectrais de alta dimensionalidade, incluindo correções radiométricas e geométricas, redução de dimensionalidade e classificação automatizada. Implementa-se um pipeline completo de processamento que integra o Método de Linha Empírica (ELM) para correção radiométrica, algoritmos de redução de dimensionalidade (PCA e t-SNE) e técnicas de deep learning (CNN e Transformer) para classificação de diferentes classes de uso da terra. A metodologia é validada através de experimentos com dados reais coletados por sensores hiperespectrais embarcados em drones, demonstrando melhorias significativas na acurácia de classificação em comparação com métodos tradicionais, alcançando precisão superior a 90\% para as principais classes de cobertura vegetal.

\textbf{Palavras-chave:} Classificação LULC, Imagens Hiperespectrais, Drones, Deep Learning, Agricultura de Precisão, Correção Radiométrica.
\end{resumo}

% Sempre deve haver um abstract:
\begin{abstract}
This work presents a methodology for Land Use and Land Cover (LULC) classification using hyperspectral images collected by drones, focusing on precision agriculture applications. The research addresses inherent challenges in processing high-dimensional hyperspectral data, including radiometric and geometric corrections, dimensionality reduction, and automated classification. A complete processing pipeline is implemented integrating the Empirical Line Method (ELM) for radiometric correction, dimensionality reduction algorithms (PCA and t-SNE), and deep learning techniques (CNN and Transformer) for classifying different land use classes. The methodology is validated through experiments with real data collected by hyperspectral sensors mounted on drones, demonstrating significant improvements in classification accuracy compared to traditional methods, achieving precision higher than 90\% for the main vegetation cover classes.

\textbf{Keywords:} LULC Classification, Hyperspectral Images, Drones, Deep Learning, Precision Agriculture, Radiometric Correction.
\end{abstract}
